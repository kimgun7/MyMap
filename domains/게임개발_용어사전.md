# 게임 개발 용어 사전

마지막 업데이트: 2026-04-21

---

## 작업 단계

| 용어 | 의미 |
|------|------|
| 그레이박싱 (Greyboxing) | 기능/구조만 잡은 초안. 비주얼 없이 틀만 잡는 단계 |
| 프로토타입 (Prototype) | 검증용 초기 버전. 방향성 확인 목적 |
| 룩뎁 (Look Dev) | 비주얼 방향 잡는 작업. Look Development의 줄임말 |
| 폴리싱 (Polishing) | 마무리 다듬기. 완성 직전 디테일 정리 단계 |
| 골드 (Gold) | 최종 완성본. 더 이상 수정 없는 상태 |

---

## 품질 / 방향

| 용어 | 의미 |
|------|------|
| 타겟룩 (Target Look) | 목표로 하는 비주얼 결과물 |
| 가이드룩 (Guide Look) | 작업 기준이 되는 샘플 |
| 레퍼런스 (Reference) | 참고 자료 |
| 톤앤매너 (Tone & Manner) | 전체적인 색감/분위기/스타일 방향 |

---

## 작업 진행

| 용어 | 의미 |
|------|------|
| 이터레이션 (Iteration) | 반복 수정 사이클. "이터레이션 돌린다" = 계속 수정하며 발전시킨다 |
| 패스 (Pass) | 한 번 훑어 작업하는 것. "FX 패스", "라이팅 패스" 등 |
| 핸드오프 (Handoff) | 작업물을 다음 담당자에게 넘기는 것 |
| 피드백 루프 (Feedback Loop) | 피드백 → 수정 → 재확인 반복 사이클 |
| 마일스톤 (Milestone) | 프로젝트 중간 목표 시점. M1, M2, M5 등 |
| 허들 (Hurdle) | 통과해야 하는 검토/심사 관문 |

---

## 팀 협업

| 용어 | 의미 |
|------|------|
| 인하우스 (In-house) | 사내 제작 (외주 반대 개념) |
| 아웃소싱 (Outsourcing) | 외부 업체에 작업 위탁 |
| 리뷰 (Review) | 작업물 검토 회의 |
| 사인오프 (Sign-off) | 담당자가 작업물 최종 승인하는 것 |

---

## FX / 이펙트 실무

| 용어 | 의미 |
|------|------|
| 나이아가라 (Niagara) | UE5의 파티클 시스템 |
| 머터리얼 (Material) | 오브젝트 표면 재질/셰이더 |
| 블렌드 모드 (Blend Mode) | 이펙트가 배경과 합성되는 방식 (Additive, Translucent 등) |
| 스폰 (Spawn) | 파티클 생성 |
| 에미터 (Emitter) | 파티클을 발생시키는 단위 |
| 루프 (Loop) | 반복 재생 |
| 원샷 (One-shot) | 한 번만 재생되는 이펙트 |
| 히트 이펙트 (Hit Effect) | 피격 시 발생하는 이펙트 |
| 트레일 (Trail) | 잔상/궤적 이펙트 |
| 디졸브 (Dissolve) | 서서히 사라지는 효과 |

---

## TA (Technical Artist) 영역

| 용어 | 의미 |
|------|------|
| 셰이더 (Shader) | GPU에서 실행되는 렌더링 프로그램 |
| UV | 3D 오브젝트 표면을 2D로 펼친 좌표 |
| 드로우콜 (Draw Call) | GPU에 렌더링 명령을 보내는 것. 줄일수록 최적화 |
| LOD (Level of Detail) | 거리에 따라 모델 해상도를 낮추는 최적화 기법 |
| 인스턴싱 (Instancing) | 동일 오브젝트를 효율적으로 대량 렌더링하는 기법 |
