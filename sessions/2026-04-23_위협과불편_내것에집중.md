# 2026-04-23 — 위협과 불편, 내 것에 집중

## 핵심 주제
- 실질적 위협 vs 불편의 기준 정의
- 유인호 패거리에 대한 감시 패턴 인식

## 결론

### 실질적 위협의 기준
위협이 성립하려면 아래 중 하나 이상 해당:
1. 권한 라인에 닿아 있음 (평가, 배정, 해고에 영향)
2. 증거 없이 윗선에 전달되어 실제 판단에 영향
3. 업무 차단 (협업 거부, 결과물 방해)
4. 구조적 고립 (정보 차단, 의사결정 배제)

→ 유인호 패거리는 현재 이 기준 어디에도 해당 안 됨

### 불편의 기준
- 인지 자원 낭비, 감정적 긴장, 신뢰 비용, 분위기 오염
- **위협은 결과를 바꾸고, 불편은 과정을 소모시킨다**

### 핵심 인식
- 유인호 패거리 근태/접속 체크 = 불안을 관리하는 방식으로 쓴 감시
- 걔네를 기준점으로 삼는 것 자체가 문제
- 걔네가 잘해서 위험한 게 아니라, 승우가 제자리면 위험한 것

## 다음 액션
- 감시 충동 올라올 때: "지금 불안한 거구나" 알아채기만 해도 충분
- 에너지 방향: 홍현우, 카잔팀, 김진홍 쪽 결과물에 집중
- 그냥 내 것에 집중하고, 학습하고, 행동하면 됨
