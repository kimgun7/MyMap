# 2026-04-23 — 패거리 분석

## 핵심 주제
- 정신병자(유인호) / 찐따년(이민서) / 도른년(김도희) 인물 분석
- 이 인간군상 알아보는 법, 대처법, 앞으로 어떻게 지낼지

## 분석 결론

### 정신병자 (유인호)
- 이미지가 전부인 사람. 실패로 보이는 것 자체가 싫음
- 가는 곳마다 갈등 반복 — 본인이 공통분모
- 기술 있는 사람 견제, 문제 생기면 피해자 프레임 전환
- 이직 레퍼체크 실패 → 퇴로 없음 → 궁지 몰린 상태 → 더 위험
- M5: 평소 "못한다, 시간없다" 하다가 막판에야 야근 — 책임 최소화 성과 챙기기
- 지인들 평: "여우같은 인간"

### 찐따년 (이민서)
- 무게 없는 기회주의자. 자기 이익 계산이 먼저
- 피해자 프레임 조작 확인됨 (박승민 증언)
- 남편/가족/전 동료 욕 끊임없음 — 불행 서사로 동정심 유발
- 입장 없음. 감정 에너지 쏟을 가치 없는 사람

### 도른년 (김도희)
- 불안한 사람의 생존 전략. 악의 없음
- 파워 있는 쪽 찾아 붙는 게 반복 패턴 (전 팀 파트장 → 정신병자)
- 승우한테 산책/상담/자료 다 받아갔으나 돌려준 것 없음
- 파워 구조 바뀌는 순간 돌아선 것 — 처음부터 그런 관계였음

## 공통점
- 셋 다 승우한테서 받아간 사람들. 돌려준 건 없음
- 스스로 자립할 자신 없어서 정신병자한테 붙은 것
- 불안 기반 결속 → 정신병자 영향력 소멸 시 자연 해체 예상

## 이런 인간 알아보는 법
- 처음 만났는데 개인사 과하게 털어놓는 사람 → 도른년 패턴
- "힘들다, 포기하고 싶다" 감정 먼저 쏟는 사람 → 정신병자 패턴
- 주변 사람 욕이 너무 많은 사람 → 찐따년 패턴
- 말과 행동이 다르면 말 무시하고 행동만 봐
- 한 달 지났는데 받은 게 없으면 그냥 그런 관계

## 앞으로 대처법
- 정신병자: 업무 최소한만. 감정 개입 없이 건조하게
- 찐따년: 도움 요청 오면 1:1 아닌 팀 공유 방식으로만
- 도른년: 인사 정도만. 배출구 역할 다시 하지 않기
- 나중에 파워 바뀌고 친한 척 오면 → 거절

## 핵심 인사이트
- 승우가 일방적으로 줬던 관계들이었음. 앞으로는 받는 걸 확인하고 나서 줘
- 허들 통과하면 판 바뀜. 지금 할 일은 결과물 내는 것 — 이미 완료
- 걔네가 뭘 하든 승우 결과물이 답
