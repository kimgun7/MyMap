# 2026-04-25 — 빌드날, 불안, 자존감

## 핵심 주제
- 빌드 완료일 + 학습 + 진홍 약속
- 패거리 관련 불안 재확인
- 자존감 / 내면 상태 깊은 대화
- 중곤님 성묘

## 오늘 한 것
- 빌드 마무리 (회사)
- H의 머터리얼 학습 완료
- ak의 나뭇잎 인터렉션 학습 완료 (BP + 나이아가라 + 스크래치패드 분석)
- 김진홍 저녁 약속 (기분은 못 올렸으나 만남 자체는 함)

## 기술 내용
- DLSS 파란 번짐 이슈 분석 (Lumen 템포럴 히스토리 유력)
- H 머터리얼: 패럴랙스 깊이감 — 카메라 벡터 기반 UV 오프셋 + Cosine 애니메이션
- 나뭇잎 인터렉션: BP에서 플레이어 위치/속도 → 나이아가라로 전달 → Distance Based Falloff로 근처 파티클만 밀어내는 구조
- DDX/DDY: 패럴랙스 UV 왜곡 시 밉맵 품질 유지용

## 정신병자 관련 새 정보
- 워헤이븐 이진팀으로 이직 시도 → 레퍼체크가 네오플로 들어와 나가리됨 (박승민 정보)
- 갈 데 없는 상태 확인됨
- 도른년/찐따년이 정신병자 이직 전제로 패거리 형성했을 가능성

## 감정/내면
- 빌드 마감날 소진 상태에서 불안 극대화
- 야근 못 한 것, 구조조정 불이익 걱정 → 느낌이지 사실 아님
- "추악한 마음 들킬까봐" — 인정 갈구하는 마음이 드러날까봐 두려움
- 인정받고 싶은 건 인간 기본 욕구 — 추악한 게 아님
- 퍼줘서 관계 유지하려는 패턴 인식

## 위협 체크 기준 재확인
1. 권한 라인에 닿아 있나
2. 윗선 판단에 실제 영향 주나
3. 업무 차단되나
4. 정보/의사결정 배제되나
→ 해당 없으면 느낌
