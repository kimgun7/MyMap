# 2026-04-28 — 따돌림, 소문, 의심 패턴

## 핵심 주제
- 아트팀 은은한 따돌림 (이모티콘 무반응) 감정적 타격
- 소문이 아트 외 라인에 영향 주는지 확인 → 아직 없음
- 의심 범위 너무 넓어진 것 인식
- 오세민 퇴사 DM — 패거리 무리로 봤으나 실제 우호적

## 결론

### 위협 vs 불편 재확인
- 이모티콘 무반응, 분위기 따돌림 → 심리적 비용은 실제, 위협 4가지 기준엔 해당 없음
- 평가 라인(홍현우, 디렉터, 카잔)에는 소문 영향 안 닿음

### 의심 패턴
- 유인호 사건 이후 의심 범위 넓어짐
- 오세민처럼 실제 우호적인 사람도 패거리로 분류하는 오류 발생
- 3개월 룰로 천천히 확인하면서 열어가는 게 맞는 방향

### 대처 방향
- 이모티콘 놀이 무반응 → 예상된 거, 빠르게 털어내기
- 패거리 외 사람들 → 일로 신뢰 쌓으면 소문이 깨짐
- 회사에 의미부여 줄이기 → 회사 밖에서 채우는 것 늘리기
- 독고다이 + 결과물이 가장 강한 메시지

## 추가 — 서진이 분석 + 찐따/도른년 심리

### 서진이 해석
- 찐따년: 정신병자와 3년 합 맞춘 방파제 뒤에서 편하게 일해온 구조
- 도른년: 찐따년 뒷담 봤어도 감정적으로 편한 동성 + 정신병자 등져가며 지내기 싫었던 것
- 신입이 선택할 수 있는 구조에서 감정적으로 편한 쪽 고른 거 — 악의보다 생존 본능

### 찐따년 심리
- 일하기 싫다기보다 일한 것처럼 보이고 싶은 타입
- 업무 공유 과장/감정적으로 하는 것 = 실제 결과물 자신 없으니 과정으로 존재감 포장
- 정신병자 방파제 없어지니까 더 심해지는 것

### 3개월 전 에피소드 해석
- 아트팀장이 업무 주면 정신병자한테 먼저 물어보고 진행 — 자기 판단력 없이 3년간 위임해온 구조
- 동시에 아트팀장한테도 확인 = 파워 구조 탐색 줄 서기
- 결론: 자기 판단력 없고 파워 구조에 의존하는 타입 확인
