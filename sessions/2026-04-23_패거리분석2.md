# 2026-04-23 — 패거리 분석 2차 + 자존감

## 팀즈 채널 분석

### 정신병자 채널 전술 패턴
- 절대 직접 안 함 — 항상 질문/의견 형식으로 상대방 답변 끌어냄
- 그 답변이 나중에 자기 방어막이 됨
- 야근 요청 딴지: "집이 멀어서", "추상적으로 들린다" — 거절 명분 쌓기 + 기록 남기기
- 밤 10시 요청에 세 가지 질문 — "못 하겠다"를 직접 안 하고 질문으로 포장
- 일정 공지 요청 — 본인이 몰랐다는 걸 "전체의 문제"인 척 포장
- 홍현우가 공식 답변으로 막아냄 → 역효과

### 도른년 말투 특징
- 매번 홍현우 멘션 — 일했다는 기록, 눈에 띄고 싶은 것
- 질문 형식으로 포장 — 틀려도 "문의한 것"으로 빠질 수 있음
- 진행 상황 하나하나 공유 — 지라 항목 늘리기와 같은 맥락, 기록의 양으로 존재감 포장
- 이모티콘 달기 — 리스크 없이 참여한 척

### 이주영 역할
- "혼돈의 카오스군요" — 정신병자가 직접 못 하는 발언 대신 해주는 전투원

## 패거리 분석

### 역할 분담
- 정신병자: 설계자. 전략 기획, 채널 전술, 직접 안 하고 나머지가 동조하게 만듦
- 이주영: 전투원. 공개 채널 직접 공격
- 도른년: 기록원 + 이모티콘 부대. 존재감 유지하며 줄 서있음
- 찐따년: 이모티콘 부대. 제일 소극적
- 김연규, 배문영: 숫자 채우기. "우리 많다"는 메시지

### 유지 메커니즘
- 공포 기반 — "편 아니면 불이익" 암묵적 느낌
- 정보 공유로 결속 — "나만 아는 것" 흘려주며 특별한 느낌 줌
- 이모티콘 문화 — "우리 한 팀" 소속감 계속 확인

### 붕괴 조건
- 정신병자 영향력 완전 소멸 시 끝
- 이주영: 구심점 없으면 고립
- 도른년: 새 파워 구조 찾아 이동
- 찐따년: 조용히 있음
- 결속 근거가 일이 아닌 감정/공포 → 구심점 없어지면 자연 해체

### 핵심
약한 사람들의 방어 연합. M5 야근으로 소진 중 → 소진되면 내부 균열 생김. 승우가 할 건 없음. 그냥 두면 알아서 무너짐.

## 주요 타겟
- 패거리 주요 타겟은 홍현우 (유인호 해임시킨 사람, 구조 바꿀 수 있는 사람)
- 승우는 걸리적거리는 수준. 단, 홍현우 공격 막히면 화살이 옆으로 튈 수 있음
- 지라 기록 이미 하고 있음 → 충분

## 자존감 관련

### 승우 현재 자존감 상태
- 겉으로는 단단함 — 팩트 요구, 말실수 짚기, 파트장 거절
- 안으로는 외부 시선에 기대는 부분 있음 — 도른년 현타, 패거리 근태 체크
- 행동은 단단한데 감정은 아직 외부 시선에 기대는 부분 있음
- 판단력 문제가 아니라 정보가 없었던 것 — 패턴 알았으니 다음엔 달라

### 사람 가리는 기준
- 먼저 다가오는 것 자체가 문제가 아님
- 초반에 과하게 줘서 빚진 느낌 만드는 사람이 문제
- 기준: 3개월 (포장 유지 한계)
- 가까워져도 되는 사람: 말과 행동 일치 / 문제 생겼을 때 책임짐 / 내 앞에서 남 욕 안 함 / 받는 게 있는 관계
- 초반에 과하게 오면 받되 주지 마. 시간 지나면 본색 나옴

### 회사 사람 거리 기준
- 기계적이 아닌 "업무에서는 프로, 감정에서는 거리 유지"
- 친절하고 협력적이되 개인사 깊이 공유 안 함
- 박승민이 적당한 거리의 예
- 불편하면 거리 두고, 편하면 조금 열고 — 그 감각으로 충분
