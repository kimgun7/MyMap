# 2026-04-21 — 커리어 목표 & 자기 믿음

## 핵심 요약
FX+TA가 직업 목표, 50살 전 자기 게임 출시가 장기 목표. 기술적 실행력은 평균 이상이나 자기 판단 신뢰 부족이 성장의 실질적 걸림돌.

## 현재 수준
- 8년+ 이펙터, 나이아가라 사용
- 기술: 평균 이상 (속도, 머터리얼, UI 지원 가능)
- 아트: 가이드룩 있으면 맞춰내고 유인호 작업 수정 수준 — 실행력 문제 아님
- 약점: 레퍼런스 없이 처음부터 방향 잡는 창조력 → 구체적으로는 연출 흐름을 시간축으로 풀어내는 것

## 뿌리 패턴
- 타이밍 재고 레퍼런스 뜯는데 구현하면서 "이게 맞나?" 반복
- 자기 기준보다 타인 시선이 판단 기준
- 작업뿐 아니라 삶 전반에 걸친 패턴 — 자기 믿음 부족
- 원인: 어릴 때부터. 불안한 가정환경, 의지할 어른 없이 성장 → 자기 판단 신뢰 훈련 못 함
- 자신감 있는 척 연기한다고 느끼지만 실제로는 해내고 있음 — 본인만 가짜라고 인식

## 원하는 상태
- 말 없이 결과로 납득시키는 것
- 작업물에 탄탄한 근거와 힘, 의지

## 결론 / 액션
- "이게 왜 좋지? 내가 쓴다면 어떻게 쓸까"를 언어화하는 훈련
- 작업 하나 끝낼 때마다 선택 이유 한 줄 설명
- 자기 기준의 언어화 → 근거 → 의지로 이어지는 구조
- 매 순간 훈련 여지 있음 확인

## 열린 질문
- 자기 믿음 부족이 관계(슬기 제외)에도 영향 주는지
- 언어화 훈련 실제로 지속되는지 다음 세션에서 확인
