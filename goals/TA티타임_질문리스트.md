# 카잔 TA (최지훈) 티타임 질문 리스트

## 목적
FX+TA 방향으로 가고 싶어서 접근 방법 파악

## 오프닝
"저 FX+TA 방향으로 가고 싶은데, TA 쪽 작업 보면서 배우고 싶어서요. 처음에 어떻게 접근하면 좋을지 여쭤봐도 될까요?"

## 질문 리스트

### 학습 관련
- TA 공부할 때 제일 도움됐던 리소스가 뭐예요? (강의, 문서, 유튜브 등)
- HLSL 직접 짜는 경우 많아요?
- 후디니 연동은 어느 수준까지 쓰세요?

### 기술 관련
- 머터리얼에서 캐릭터 데이터 받아올 때 제일 많이 쓰는 방식이 뭐예요?
- 나이아가라랑 머터리얼 연동할 때 병목 생기는 부분이 어디예요?
- PCG 쓰실 때 이펙트랑 연동하는 경우 있어요?

### 실무 관련 (제일 실용적)
- 이펙터랑 협업할 때 제일 많이 요청받는 게 뭐예요?
- TA 포지션에서 이펙터한테 기대하는 기술 수준이 있어요?

## 메모
- 다 물어보지 말고 분위기 보면서 2~3개만
- 나뭇잎/물 인터렉션 구현 방식은 리버스엔지니어링으로 직접 파보기로 함
